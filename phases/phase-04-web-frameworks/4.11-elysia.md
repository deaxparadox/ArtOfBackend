## 1. What Is Elysia?

Elysia (2023, SaltyAom) is a TypeScript web framework built specifically for **Bun** — the fast JavaScript runtime. It achieves top benchmark numbers by leveraging Bun's performance and TypeScript's type inference to generate optimized validation code at compile time.

**Key features:**
- End-to-end type safety (request → response types inferred automatically)
- Runtime type validation built in (no Zod needed separately)
- Fastest benchmarks of any TypeScript framework

```typescript
import { Elysia, t } from 'elysia';
import { jwt } from '@elysiajs/jwt';
import { swagger } from '@elysiajs/swagger';

const app = new Elysia()
    .use(swagger())       // auto-generates OpenAPI docs
    .use(jwt({ name: 'jwt', secret: process.env.JWT_SECRET! }))

    // Types are inferred end-to-end:
    .get('/users/:id', async ({ params: { id } }) => {
        const user = await db.getUser(parseInt(id));
        if (!user) throw new Error('Not found');
        return user;
    }, {
        params: t.Object({ id: t.String() }),
        response: t.Object({ id: t.Number(), name: t.String() }),
    })

    .post('/users', async ({ body }) => {
        return db.createUser(body);
    }, {
        body: t.Object({
            email: t.String({ format: 'email' }),
            name: t.String({ minLength: 1 }),
        }),
    })

    .listen(3000);

console.log(`Running at ${app.server?.hostname}:${app.server?.port}`);
```

**When to choose Elysia:** New Bun-based projects where you want maximum TypeScript type safety and the fastest possible framework performance. **Not recommended for Node.js** — Elysia's Bun-specific optimizations don't transfer.
