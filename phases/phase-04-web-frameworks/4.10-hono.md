## 1. What Is Hono?

Hono (2022, Yusuke Wada) is an ultrafast, lightweight web framework that runs on any JavaScript runtime: Node.js, Bun, Deno, Cloudflare Workers, AWS Lambda, Vercel Edge, Fastly Compute@Edge.

**Why it matters:** Most frameworks are tied to one runtime. Hono's portability means the same code runs at the CDN edge (Cloudflare Workers, <1ms globally) or on a Node.js server — without changes.

```typescript
import { Hono } from 'hono';
import { zValidator } from '@hono/zod-validator';
import { z } from 'zod';

const app = new Hono();

app.use('*', async (c, next) => {
    const start = Date.now();
    await next();
    c.res.headers.set('X-Response-Time', `${Date.now() - start}ms`);
});

const createUserSchema = z.object({
    email: z.string().email(),
    name: z.string().min(1),
});

app.get('/users/:id', async (c) => {
    const id = c.req.param('id');
    const user = await db.getUser(parseInt(id));
    if (!user) return c.json({ error: 'Not found' }, 404);
    return c.json(user);
});

app.post('/users', zValidator('json', createUserSchema), async (c) => {
    const data = c.req.valid('json');
    const user = await db.createUser(data);
    return c.json(user, 201);
});

// Deploy to Cloudflare Workers:
export default app;

// Deploy to Node.js:
import { serve } from '@hono/node-server';
serve({ fetch: app.fetch, port: 3000 });
```

**When to choose Hono:** Edge functions (Cloudflare Workers, Vercel Edge), serverless APIs, or when you want to run the same code across runtimes.
